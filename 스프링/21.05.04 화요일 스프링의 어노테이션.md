# 21.05.04 화요일 스프링의 어노테이션

## 어노테이션이란?

- 컴파일러가 무시하지 않는 주석

- `@`를 이용하여 표시한다

- ```java
  Animal {
      run();
  }
  ```

- ```java
  Dog 상속 Animal
      @override
      run();
  ```

  - Dog를 컴파일 할 때 Animal에 run이라는 메서드가 있는지 확인함
  - 덮어씌우는 용도로 사용



## 스프링에서 어노테이션을 이용하는 법

- 객체의 생성에 이용함

- ```java
  @Component -> 클래스를 메모리에 로딩
  @Autowired -> 로딩된 객체를 해당 변수에 집어넣음
  ```

  - ```java
    @Component // 메모리에 A 객체를 추가함
    Class A {
        
    }
    
    Class B {
        A a = new A(); // 메모리의 새 공간에 A 객체를 추가함
        @Autowired
        A a; // 메모리에 기존에 추가되어 있는 A 객체를 가져옴
        // 없을 경우 null이 된다
    }
    ```
